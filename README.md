## About The Project

This a recipe finder application that searches over 100k recipes from all over the world. This application also has features like shopping list, favorite recipes, 
ingredient list and serving time for better user experience.

### Built With

* HTML
* CSS
* JavaScript
* Ajax

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You don't need any prerequisites to run this application

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/uvrays123/Forkify-Recipe-Finder
   ```
2. Install the NPM packages
   ```
   npm install --save
   ```

3. start the application
   ```
   npm start
   ```
